# J.K. Genesis Mitchell Site

This is a deploy-ready Next.js website package for J.K. Genesis Mitchell.

## Quick start

1. Install Node.js 18 or newer.
2. Open a terminal in this folder.
3. Run:
   npm install
4. Then run:
   npm run dev
5. Open:
   http://localhost:3000

## Deploy to Vercel

1. Create a GitHub repository.
2. Upload all files in this folder.
3. Go to Vercel.
4. Import the GitHub repository.
5. Click Deploy.

## Important before going live

The contact form currently uses a placeholder Formspree action:

https://formspree.io/f/your-form-id

Replace that with your real Formspree endpoint before publishing.
